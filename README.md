## 简介：
这个项目可以让你快速的创建时间线，你只需要提供一个csv数据文件，即可生成一个基于html的时间线可视化图！

## 安装指南：
安装requirements.txt所需库
```bash
pip install -r requirements.txt
```

## 数据格式（参考sampledata.csv）： 
year,title,subtitle,description  
1811,初步发现硅,盖-吕萨克，泰纳  
1824,纯化硅,永斯·雅各布·贝采利乌斯  
1854,制备出硅晶,德维尔
